# Banking Transaction Simulator

A full-stack banking transaction simulator built with Java Spring Boot and MySQL.

## Tech Stack
- Java 17
- Spring Boot 3.2
- MySQL 8.0
- HTML/CSS/JavaScript
- Maven

## Features
- Account Management (CRUD)
- Deposit, Withdraw, Transfer
- Transaction History
- Mini Statement
- Balance Alert Tracker
- Reports Generation
- Search by Name
- Account Types (SAVINGS/CURRENT/CHECKING)

## Setup
1. Create MySQL database: CREATE DATABASE bankingdb;
2. Update application.properties with your MySQL password
3. Run: ./mvnw spring-boot:run
4. Open: http://localhost:8080
